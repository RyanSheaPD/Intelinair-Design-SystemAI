AgMRI Weather — hover states

One page for design export: quick links (open or copy URLs) and frozen hover slabs (each iframe loads agmri-weather.html with figmaHover=…). Real :hover is not captured by static export tools.

Serve over HTTP (e.g. python3 -m http.server from the repo root), then open pages/weather/weather-hover-states.html. Iframes will not load from file://. Manifest: data/weather-hover-states.json · frozen styles: pages/css/weather-hover-export.css (loaded by agmri-weather.html). After editing the manifest, reload this page (no HTML regen required). Field detail modal (Figma): use figmaHover=field-detail-sheet&field=8 on map view — the bottom sheet opens on load (see manifest rows 01–02).

GW / spray gallery: weather-gw-spray-tooltip-options.html
