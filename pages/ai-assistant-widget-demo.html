AI assistant widget

Floating control for agronomy / ops apps: users tap Assistant to open the chat composer (type to the LLM), or tap the microphone for hands-free input (speech-to-text in the browser where supported).

In native iOS or Android apps, use platform STT (iOS Speech framework / SFSpeechRecognizer; Android SpeechRecognizer or on-device ML). Port these layout and color tokens for parity; avoid relying on the Web Speech API inside WebViews in production unless you explicitly support it.

  • Assistant — opens the panel focused on the text field.
  • Mic (on the strip or in the composer) — starts listening; transcribed text fills the field (demo uses the Web Speech API when available).
  • Send — appends a user bubble (no real LLM in this static demo).
