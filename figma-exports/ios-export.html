HTML → Figma export Figma’s HTML-to-Design capture only runs over http://localhost (or 127.0.0.1), not file://.
  1. From the design-system repo root, run: python3 -m http.server 8901
  2. In Cursor, ask to export this page with the Figma MCP — it generates a one-time URL with #figmacapture=… that opens in your browser and sends the layout to Figma.
  3. Open: http://127.0.0.1:8901/figma-exports/ios-export.html (add the capture hash when the tool provides it).
The capture script is already loaded in the page head; the hash activates it.
